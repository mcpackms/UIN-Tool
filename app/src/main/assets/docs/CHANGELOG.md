# UIN Tool 更新日志

本文档记录了 UIN Tool 的所有重要版本更新和功能变化。

---

## [1.1.0] - 2026-06-07

### 🎉 新增功能

#### 分类管理增强
- **添加分类功能**：用户可以在插件详情中直接创建新分类
- **删除分类功能**：支持删除自定义分类，删除后该分类下的插件自动移动到"未分类"
- **分类统计显示**：在分类列表中显示每个分类下的插件数量
- **快速创建分类**：在修改插件分类时，可选择"新建分类"快速创建

#### 工具页面优化
- **长按菜单**：在工具页面的插件列表和网格视图中，长按插件会显示详情弹窗
- **插件详情弹窗**：包含以下功能：
  - 运行插件
  - 创建桌面快捷方式
  - 修改插件分类
  - 卸载插件
- **统一交互体验**：工具页面与管理页面的插件操作体验保持一致

#### 小部件系统重构
- **统一配置逻辑**：合并冗余的小部件配置代码
- **优化刷新机制**：小部件刷新更加稳定可靠
- **1x1 快捷方式小部件**：支持绑定单个插件作为桌面快捷方式

#### 错误处理增强
- **统一错误处理**：新增 ErrorHandler 工具类，统一管理异常记录
- **完善日志系统**：所有异常都通过 LogUtils 记录到文件
- **安全执行包装器**：提供 safeExecute 方法，自动捕获异常并返回默认值

#### 文档系统增强
- **新增更新日志查看**：在关于页面中可以查看完整的版本更新日志
- **Markdown 渲染优化**：改进文档显示效果

### 🔧 改进优化

- 优化 PluginManager 中的分类管理方法
- 改进日志记录系统，增加更多上下文信息
- 优化插件加载性能
- 改进 WebView 配置，提升兼容性
- 优化 UI 响应速度

### 🐛 问题修复

- 修复小部件配置页面可能崩溃的问题
- 修复插件分类更新后列表不刷新的问题
- 修复批量导入插件时的进度显示问题
- 修复深色模式下部分颜色显示异常的问题
- 修复 Web 插件生命周期事件传递问题

### 📝 开发相关

- 完善开发文档
- 添加代码注释
- 优化项目结构

## [1.0.0] - 2026-06-06

### 🎉 首次正式发布

#### 核心功能

##### 插件系统
- **插件框架**：完整的插件化架构，支持动态加载和运行插件
- **双模式插件**：
  - **原生插件**：通过 DexClassLoader 加载 Java 代码实现的插件，性能最优
  - **Web 插件**：通过 WebView 加载 HTML/CSS/JS 实现的插件，支持热更新
- **插件管理**：
  - 导入插件（支持单文件、批量导入、插件集 ZIP 导入）
  - 导出插件（批量导出为 ZIP 文件）
  - 卸载插件
  - 插件详情查看
  - 插件签名验证（SHA-256 哈希校验）
- **插件权限管理**：
  - 细粒度的插件权限控制
  - 支持 30+ 种权限类型
  - 插件权限独立存储

##### 开发工具
- **插件创建向导**：
  - 原生插件向导：自动生成实现 PluginInterface 接口的 Java 代码
  - Web 插件向导：
    - 完整模板（包含示例 HTML/CSS/JS）
    - 空白模板（基础框架）
    - 导入已有 Web 项目（ZIP 格式）
- **代码编辑器**：
  - 内置于应用的代码编辑器
  - 支持语法高亮
  - 文件管理（添加/删除文件）
  - 实时保存
- **本地编译工具**：
  - 集成 ECJ 编译器（Eclipse Compiler for Java）
  - 集成 D8 编译器（Android DEX 编译器）
  - 支持 Java 源码到 DEX 的转换
  - TPK 打包工具

##### 应用管理
- **权限管理**：
  - 一键授权
  - 权限分类展示（存储、网络、相机、位置等 13 个分类）
  - 特殊权限引导（悬浮窗、修改系统设置等）
  - 插件权限独立管理
- **备份恢复**：
  - 备份全部插件和配置
  - 从备份恢复
  - 批量备份管理
  - 备份文件导出/导入
- **日志查看**：
  - 运行日志实时查看
  - 崩溃日志自动捕获
  - 日志导出功能
  - 清空日志功能

##### UI 个性化
- **主题定制**：
  - 自定义主题色
  - 自定义强调色
  - 自定义成功/警告/错误色
  - 实时预览
- **形状定制**：
  - 自定义圆角大小（小/中/大/超大）
  - 按钮圆角
  - 卡片圆角
  - 弹窗圆角
- **深色模式**：完整的深色模式支持
- **配置导入导出**：支持 UI 配置的导入和导出

##### 桌面小部件
- **列表小部件**：
  - 可配置显示 3 个常用插件
  - 点击直接运行插件
  - 支持刷新按钮
- **1x1 快捷方式小部件**：
  - 绑定单个插件
  - 点击直接运行
  - 可配置打开主界面

##### 无障碍服务
- **无障碍功能**：
  - 模拟返回操作
  - 模拟主页操作
  - 模拟最近任务
  - 通知栏操作
  - 快速设置操作
  - 锁屏操作
  - 截图操作（Android 9+）

##### 文档系统
- **内置文档**：
  - 开发文档（README.md）
  - 使用帮助（Help.md）
  - 关于页面（About.md）
  - 贡献者名单（CONTRIBUTORS.md）
  - 更新日志（CHANGELOG.md）
- **Markdown 渲染**：
  - 支持标题、列表、代码块、表格
  - 支持目录锚点跳转
  - 支持代码复制功能
  - 美观的样式设计

### 🛠️ 技术特性

- **最低 Android 版本**：API 21 (Android 5.0)
- **目标 Android 版本**：API 33 (Android 13)
- **架构**：MVVM + Fragment 单 Activity 架构
- **日志系统**：完整的文件日志系统，自动截断
- **崩溃捕获**：全局异常捕获，自动写入日志
- **签名验证**：插件 SHA-256 签名验证
- **权限管理**：运行时权限 + 特殊权限管理
- **资源加载**：插件独立 AssetManager 和 Resources

### 📦 支持的文件格式

- **TPK**：UIN Tool 插件包格式（基于 ZIP）
- **ZIP**：插件集打包格式
- **PNG**：插件图标格式（推荐 128x128）
- **JSON**：插件配置文件格式

### 📝 示例和模板

- 原生插件示例代码
- Web 插件示例（HTML/CSS/JS）
- 完整的开发文档
- README 模板

### 🔒 安全性

- 插件签名验证（SHA-256）
- 插件权限独立管理
- WebView 安全配置
- 文件访问权限控制

### 🌐 多语言支持

- 中文（简体）

### 📱 兼容性

- 支持桌面小部件（Android 4.2+）
