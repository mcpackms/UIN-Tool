# 关于 UIN Tool

## 应用简介

UIN Tool 是一个强大的 Android 插件化框架应用，允许用户动态加载和运行第三方插件。无论是原生 Java 插件还是 Web 技术栈（HTML/CSS/JS）的插件，都能在 UIN Tool 中无缝运行。它提供了一个完整的插件生态系统，包括插件开发、管理、运行和权限控制等功能。

### 核心理念

- **开放**：任何人都可以开发插件，支持原生 Java 和 Web 技术栈
- **安全**：插件权限独立管理，支持签名验证，防止恶意插件
- **高效**：原生性能，Web 插件支持热更新，无需重新编译
- **易用**：可视化开发向导，无需复杂配置即可创建插件
- **灵活**：支持网格/列表视图切换，支持分类管理

### 版本历史

#### v2.8.0 (2026年6月8日)

**🎉 重大更新：应用内更新与版本管理**

**新增功能：**
- **应用内更新检查**：从 GitHub Releases 自动检测最新版本
- **版本列表浏览**：支持查看所有历史版本，选择任意版本下载
- **应用内下载**：直接在应用内下载 APK，无需跳转浏览器
- **下载进度显示**：实时显示下载百分比和速度
- **后台下载支持**：支持后台下载，下载完成后通知栏提醒
- **一键安装**：下载完成后直接调用系统安装器
- **更新日志查看**：在版本详情中查看完整更新日志
- **版本比较**：自动标记比当前版本更新的版本

**优化：**
- 移除 SSL 证书检查，解决部分设备的连接问题
- 优化 Fragment 生命周期，防止 Context 泄漏
- 增强异常处理，提升稳定性
- 优化下载线程管理，避免内存泄漏
- 改进对话框显示逻辑，防止 BadTokenException
- 优化文件命名规则，避免路径问题

**修复：**
- 修复 Fragment 销毁后显示 Toast 导致的崩溃
- 修复 Activity 销毁后显示对话框导致的崩溃
- 修复 FileProvider 路径配置问题
- 修复下载过程中 Activity 销毁导致的崩溃
- 修复版本解析错误问题

#### v2.6.0 (2026年6月7日)

**🎉 重大更新：插件功能全面增强**

**新增功能：**
- **Web 插件网络请求**：Web 插件可通过 `UINPlugin.httpGet()` 和 `UINPlugin.httpPost()` 发起 HTTP 请求
- **Web 插件文件系统**：支持读写、删除、列出插件目录内的文件
- **Web 插件传感器**：Web 插件可访问加速度计、陀螺仪、磁场、光线、接近、压力传感器
- **插件搜索功能**：在仓库页面和工具页面支持按名称、ID、描述、作者搜索插件
- **GitHub 镜像加速**：内置多个镜像站，自动选择最快镜像加速插件下载
- **插件详情弹窗**：点击插件卡片显示完整信息（版本、大小、更新日志等）
- **自定义 Dialog**：原生插件使用自定义 Dialog 避免主题兼容性问题

**优化：**
- 插件加载稳定性大幅提升
- WebView 配置优化，提升兼容性
- 文件系统操作增加路径安全检查
- 传感器数据实时显示优化
- 插件列表加载速度优化
- 错误处理和日志输出优化

**修复：**
- 修复 AlertDialog 在部分设备上的崩溃问题
- 修复 PluginContext 主题属性解析错误
- 修复文件写入后列表不刷新问题
- 修复 Fragment 菜单不显示问题

## 常见问题

### Q: 如何安装插件？
**A**: 有三种方式：
- 从「仓库」页面直接浏览和安装官方插件
- 将 `.tpk` 文件导入手机，在「管理」→「插件管理」中导入
- 通过批量导入或插件集导入多个插件

### Q: Web 插件和原生插件有什么区别？
**A**: Web 插件使用 HTML/CSS/JS 开发，无需编译，修改后即时生效，支持通过网络请求、文件系统、传感器等 JS API 调用原生功能；原生插件使用 Java 开发，性能更好，但需要编译。

### Q: 插件签名验证失败怎么办？
**A**: 可以在「管理」→「开发者选项」中开启「忽略签名验证」（仅用于测试）。

### Q: 如何开发自己的插件？
**A**: 点击底部「开发」→「创建新插件」，按照向导操作即可。Web 插件支持完整模板和空白模板两种方式。完成后可发布到 GitHub 官方仓库。

### Q: 如何备份插件和数据？
**A**: 在「管理」→「备份恢复」中点击备份按钮。

### Q: 应用崩溃了怎么办？
**A**: 应用会自动捕获崩溃日志，下次启动时会自动打开日志查看器。你可以在「管理」→「运行日志」中查看和导出日志。

### Q: 如何查看版本更新内容？
**A**: 在「管理」→「查看文档」→「更新日志」中查看。

### Q: 如何管理插件分类？
**A**: 在插件详情弹窗中点击「修改分类」，可以新建分类或选择已有分类。

### Q: 如何将插件发布到官方仓库？
**A**: 参考开发文档中的「发布到插件仓库」章节，按照要求在 GitHub 上创建仓库并发布 Release。

### Q: 仓库页面加载失败怎么办？
**A**: 检查网络连接，下拉刷新重试。系统会自动切换镜像站加速下载。也可以在「管理」→「开发者选项」中检查网络设置。

### Q: Web 插件如何发起网络请求？
**A**: 使用 `UINPlugin.httpGet(url, callback)` 和 `UINPlugin.httpPost(url, data, callback)` 方法，支持异步回调。

### Q: Web 插件如何读写文件？
**A**: 使用 `UINPlugin.writeFile(name, content)`、`UINPlugin.readFile(name)`、`UINPlugin.deleteFile(name)`、`UINPlugin.listFiles(dir)` 等方法。

### Q: Web 插件如何访问传感器？
**A**: 使用 `UINPlugin.startSensor(type, callback)` 启动传感器，数据会通过回调实时返回。支持加速度计、陀螺仪、磁场、光线、接近、压力传感器。

### Q: 如何检查应用更新？
**A**: 在「管理」页面点击「检查更新」卡片，系统会自动检测最新版本并显示版本列表。

### Q: 下载的 APK 保存在哪里？
**A**: 下载的 APK 文件保存在 `/storage/emulated/0/UIN_Tool/downloads/` 目录下。
