## 如何成为贡献者

我们欢迎各种形式的贡献！

### 代码贡献

1. Fork 项目仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request

### 报告 Bug

1. 确认 Bug 未被报告过
2. 使用最新版本测试
3. 提交详细的问题描述
4. 提供复现步骤

### 建议新功能

1. 在 Issues 中描述功能
2. 说明使用场景
3. 讨论实现方案

### 完善文档

1. 修正错别字
2. 补充说明
3. 添加示例
4. 翻译文档

### 开发插件

1. 使用 UIN Tool 开发向导
2. 开发有创意的插件
3. 分享给社区
