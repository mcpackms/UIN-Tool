# ==================== 配置说明 ====================
# 
# 镜像站分类及数量统计：
# 1. 直接访问型：13个 - 完全替代GitHub进行网页浏览
# 2. 文件加速型：28个 - 专用于下载Release、ZIP包等大文件
# 3. 高校及云厂商：7个 - 清华、南大、中科大、浙大、上交等
# 4. FastGit系列：9个 - FastGit全球加速服务
# 5. ghproxy系列：7个 - 专业GitHub代理服务
# 6. gitclone系列：4个 - Git克隆加速专用
# 7. 其他可用镜像：15个 - 实测可用的补充镜像站
# 8. 备用镜像站：13个 - 可用性较高的备选
# 9. 补充镜像站：4个 - 凑齐100个
# 
# 总计：100个镜像站
# 
# 使用建议：
# - 日常浏览代码：优先使用 bgithub.xyz、kgithub.com、hub.fastgit.org
# - 下载Release包：优先使用 ghproxy.net、gh-proxy.com、moeyy.cn/gh-proxy
# - 学术场景：优先使用 mirrors.tuna.tsinghua.edu.cn
# - 克隆仓库：可使用 gitclone.com 或配置全局镜像
# - Git Clone加速：git clone https://ghproxy.net/https://github.com/用户/仓库.git
# 
# 注意事项：
# - 镜像站不支持登录账户、Star、Fork、PR等需身份验证的操作
# - 私有仓库无法通过镜像访问
# - 部分镜像站可能存在访问限制，建议多备选几个
# - 镜像站可能因网络策略变化而失效，本列表会持续更新
